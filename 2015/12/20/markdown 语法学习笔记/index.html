markdown 语法学习笔记

发表于 2015-12-20   |   分类于 markdown
markdown语法很简洁明了。一边学习一边总结markdown语法。
建议所有计算机专业都自学markdown语言
其实也就一个电影的功夫

博文中添加目录

# 这是一级目录
## 这是二级目录
### 这是三级目录

  有些博客主题会自动生成文章目录，前提是你要在文章的目录上标上#符号。你看到的#是写在目录前面的，#的个数代表目录的等级数。如果你用恰巧也用macdown写博文的话，可以把光标移到你认为合适的地方，按住command + 1到 command + 6,分别对应了一到六级目录。
  从NEXT主题自动生成的目录来看，1、2、3这种为一级目录；1.1、1.2、1.3这种为二级目录；1.1.1、1.1.2、1.1.3为三级目录，以此类推。

首行缩进

&emsp;&emsp;
  如果博文是中文的，最好还是做 首行的缩进，这样才不会对阅读带来视觉的压迫感。在段落首字母前加入。记不住怎么办？ems能记住吧？
  HTML中的缩进其实还有&ensp、&nbsp;、&thinsp;,以后再说吧。

文字居中对齐

  首先markdown不支持 文字居中对齐。但是我们依然可以使用HTML语言来进行设置。

  • 正文使用<center>文字</center>
    *标题使用<h1 style="text-align:center">文字<h1>

文章插入 阅读全文

  在你想在主页中隐藏的文章上面插入 <!-- more --> 就能隐藏下面的文章而让 主页变得很整洁。感受一下，多么有力量的语言啊。

文字加粗&斜体

文字两边个一个 * 号为斜体，两个为加粗，三个为斜体并加粗。实例如下：
*斜体*   斜体
**加粗**  加粗
***斜体并加粗*** 斜体并加粗

#插入优酷视频
打开想链接的优酷上的视频，点击视频下面的分享，会有html链接。
<embed src="http://player.youku.com/player.php/... ></embed>
把这个链接直接复制到.md 文件中即可。亲测。
