ObjectiveC笔记一

发表于 2014-02-02   |   分类于 IOS

id 类型      NSObject 或 void

nil –> null 目标指针

类 +
对象 -  （实例方法）

实例方法总是可以直接访问它的实例变量。但是 类方法不能。因为它只处理类本身，并不处理类的任何实例。 对象是类的独特表示。

.h   类，函数申明 @interface、@end
.m   类的具体实现 @implementation、@end

@interface： 描述类、类的数据成分以及类的方法。
@implementation： 实现 方法 的实际代码。

创建对象
Dog *dog = [Dog alloc];

初始化(构造函数)
[dog init];

销毁
[dog release];

@public / @protected (缺省)/ @private

函数全是public
函数定义在{ }之间

接口声明 即 指针 又 引用
Dog myDog;
通过 myDog -> dog 或 myDog.dog访问

函数定义

属性： @interface{ 和 } 之间
方法： -（int）f: (int)X;
-：对象的方法
+: 类的方法
():是返回值或参数
参数由 : 来分开
ps: : 后面必须有参数

多参数：
-（void）insertObject:(int)value :(unsigned int)index

第二个不带标签label，即 匿名。
函数名： insertObject ：：

调用：
[obj insertObject:100 :20];
[obj    :101 :21];

-（void）insertObject:(int)value atIndex:(unsigned int)index

第二个的标签为atIndex
函数名：insertObject：atIndex:

调用：
[obj insertObject:100 atIndex:20];
[obj insertObject:101 atIndex:21];

Message 消息机制：
message可以认为是一个函数
[object message] 或 object.message

方法体外 -> 实例变量 –> 不可以给初值 （成员变量）
方法体内 -> 局部变量 –> 默认为 0

数据类型 符号
int %d、%h
short int %hd、%h
long int %ld、%l
unsigned int %u
unsigned short %lu
unsigned long %lu
float &f
double %f
long double %lf
char %c

ID 类型：
number声明为id类型 -> id number

数据类型转换：

容量小 -> 容量大
(byte、short、char) int -> long -> float -> double

容量大 -> 容量小 强制

break: 在循环体中，强制退出
continue: 在循环体中，终止某一次循环
